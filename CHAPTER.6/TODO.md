# 로또 번호 생성기 프로젝트

## 📋 프로젝트 요구사항

### 기술 스택
- 순수 HTML, CSS, JavaScript 사용
- 외부 라이브러리 사용 금지

### 주요 기능

#### 1. 로또 번호 생성
- 버튼 클릭 시 1~45 범위의 숫자 중 중복되지 않는 6개의 번호 생성
- 생성된 번호는 오름차순으로 정렬하여 표시

#### 2. 히어로 섹션 애니메이션
- 1~45까지의 숫자가 적힌 45개의 공이 에어볼 추첨기에 들어가있는 형식의 애니메이션
- CSS 애니메이션을 활용한 자연스러운 모션 구현

#### 3. 히스토리 기능
- 생성된 로또 번호를 히스토리 섹션에 누적하여 표시
- 로컬 스토리지를 활용한 데이터 저장 (선택사항)
- 최신 생성 번호가 상단에 표시
